# Calculator App

A simple Android Calculator application developed using Java and Android Studio.

## Features

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Clear (C) button
* Expression display
* Result display
* Input validation
* Division by zero handling
* Custom Blue Sky UI theme
* Reusable button styles

## Technologies Used

* Java
* Android Studio
* XML Layouts
* Android Views (TextView, Button)
* Styles and Resources

## How to Run

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the project in Android Studio.

3. Wait for Gradle Sync to complete.

4. Run the application on:

   * Android Emulator
   * Physical Android Device
